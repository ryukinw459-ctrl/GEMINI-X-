# Gemini Apex V38 - Biological Behavior Engine

Sistema de IA avançado para Roblox que simula limitações biomecânicas e cognitivas humanas.

### ✨ Funcionalidades Core:
- **Metabolismo Cognitivo**: Gestão de fadiga acumulativa e recuperação não-linear.
- **Warmup Orgânico**: Transições suaves de performance (2s de aceleração).
- **Entropia de Comportamento**: Ruído e distrações persistentes (Inércia de erro).
- **Burnout Bifásico**: Ciclos de exaustão com chance de recaída.
- **Independência de FPS**: Lógica estocástica baseada em seeds, estável em qualquer framerate.

### 🚀 Como usar:
Instale o `ModuleScript` em `ServerStorage` e chame via `require`.
